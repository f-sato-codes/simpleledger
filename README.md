# SimpleLedger

## 概要
**SimpleLedger** は、月ごとの収支をシンプルに管理できる家計簿アプリです。  
「必要な機能に絞り、後から拡張しやすい設計」を意識して個人開発しました。

Laravel を用いて、**設計 → 実装 → UI改善**まで一通り行っています。

---

## 主な機能

- ユーザー認証（Laravel Breeze）
- 収支の登録・編集・削除（CRUD）
- 月切り替えによる収支一覧表示
- 月次サマリー（収入・支出・収支）
- カテゴリ管理（登録・削除）
  - 使用中のカテゴリは削除不可
- ログアウト機能

---

## 画面構成

### 収支一覧画面
- 月切り替え（前月・翌月）
- 月次サマリー表示
- 取引一覧（カテゴリ・金額・編集・削除）
- 金額は **収入＝青／支出＝赤** で視認性を向上

### カテゴリ管理画面
- カテゴリの追加
- 使用中カテゴリの削除制御

## 設計・実装で工夫した点

### 1. 共通レイアウトの分離
- `layouts/app.blade.php` と `navigation.blade.php` に分離
- 各画面は `@section('content')` のみを記述
- 画面追加時の修正箇所を最小限に抑える構成

### 2. ORM（Eloquent）の理解を重視
- `Transaction` と `Category` を **belongsTo / hasMany** で明確に定義
- SQL の JOIN を Eloquent のリレーションとして説明できる構成

### 3. 業務ルールを Controller で制御
- 使用中カテゴリの削除不可判定を Controller に実装
- UIだけに依存しない、安全なデータ管理を意識

### 4. UI・可読性の改善
- 画面を中央寄せし、カード型レイアウトを採用
- 金額は右揃え＋色分け
- 視線の流れを  
  **月 → サマリー → 明細 → 操作**  
  となるよう調整
